============
Installation
============


Stable release
--------------

To install ChatGPT for gaming research, run this command in your terminal:

.. code-block:: console

    $ pip install chatgpt_gaming_framework

This is the preferred method to install ChatGPT for gaming research, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for ChatGPT for gaming research can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Razikus/chatgpt_gaming_framework

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Razikus/chatgpt_gaming_framework/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Razikus/chatgpt_gaming_framework
.. _tarball: https://github.com/Razikus/chatgpt_gaming_framework/tarball/master
